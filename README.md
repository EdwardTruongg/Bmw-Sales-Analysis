# Bmw-Sales-Analysis

## Overview
This project analyzes simulated global BMW vehicle sales data from 2015 to 2024. 
SQL was used to clean and prepare the data, and Tableau was used to build an 
interactive dashboard exploring sales trends, regional performance, model 
popularity, and the shift toward electric and hybrid vehicles over time.

## Dataset
Source: Kaggle — BMW Global Sales Dataset 
Link: https://www.kaggle.com/datasets/payaldhokane/bmw-global-sales-and-market-data
Note: This dataset is synthetically generated for educational and analytical 
purposes and does not represent official BMW sales figures.


## Questions Explored
- How did total BMW sales change year over year?
- Which models sold the most units globally?
- How has the mix of fuel types shifted from 2015 to 2024?
- What percentage of sales were Electric or Hybrid each year?
- Which regions had the highest sales volume?

## Key Findings
- China led all regions in total units sold
- The X1 was the best selling model across the full dataset
- Electric and Hybrid combined consistently made up over 50% of annual sales
- Diesel sales declined steadily as Electric vehicles grew year over year
